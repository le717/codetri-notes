[meta] title = #vss365today - Site Paused date = 2023-02-08T00:00:00-05:00 url = vss365today-site-pause [endmeta]

Due to the Twitter API becoming a paid access API some time tomorrow, with pricing still to be announced, #vss365 today has been forced to stop auto-updating itself with the latest #vss365 Prompt. Effective immediately, no new Prompts, notification emails, or spreadsheet archives will be generated or made available. Additionally, new notification email registrations have been disabled.

All existing Prompts, as well as the newest available spreadsheet archive, will remain online and unchanged. All other site aspects will also remain available.

This pause will remain in effect until Twitter API pricing has been announced and a suitable maintenance plan is developed. Should the pause be removed, all missing Prompts will be loaded into the site to ensure a complete archive is available.

It is desired that this pause is temporary and will not last for an extended time.

Additional updates will be made available through this site and @cely717.

Blog post with additional information and personal notes: https://blog.codetri.net/vss365today-site-pause/.
