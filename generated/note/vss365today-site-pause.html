#vss365today - Site Paused

Published 2023-02-08T00:00:00-05:00

Due to the Twitter API becoming a paid access API some time tomorrow, with pricing still to be announced, #vss365 today has been forced to stop auto-updating itself with the latest #vss365 Prompt. Effective immediately, no new Prompts, notification emails, or spreadsheet archives will be generated or made available. Additionally, new notification email registrations has been disabled.

All existing Prompts, as well as the newest available spreadsheet archive, will remain online and unchanged. All other site aspects will also remain available.

This pause will remain in effect until Twitter API access pricing has been announced and a suitable plan going forward is developed. Should the pause be removed, all missing Prompts will be loaded into the site to ensure a complete archive is available.

It is desired that this pause is temporary and will not last for an extended time.

Additional updates will be made available through this site and @cely717.

Blog post with additional information: https://blog.codetri.net/
